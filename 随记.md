# 虚拟机桥接网络

将虚拟机视为网络中的一台独立计算机。本地物理网卡 和 虚拟网卡 通过 虚拟交换机进行桥接，那么物理网卡和虚拟网卡相当于处于一个网段，就像它们是同一个局域网中一样。





# wifi中的三种模式

- 基础设施：你连别人

- 自组网：两个设备之间直接通信，而无需通过基础设施

- 接入点：别人连你



# make和make install

- make

根据Makefile中的指令进行编译，生成目标文件，但不安装	

> Makefile定义 
>
> - 编译源代码 
> - 链接库文件 
> - 生成可执行文件

- make install

将文件 安装到 指定目录，方便全局访问
